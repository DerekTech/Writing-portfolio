What is Laravel?

Laravel is an open-source PHP framework that makes building websites and apps easier. It works well with MySQL databases and is easy to connect.

In addition to providing you with the tools to build a secure website, Laravel comes with a host of features that will make your web development task faster. Saving you a lot of time, especially if you're building your website from scratch.

Features

Laravel offers so many interesting features, such as:

  • Blade, a templating engine for creating dynamic HTML pages
  • An easy-to-setup authentication system
  • Eloquent Models that eliminate the need to write a load of SQL to query a MySQL database
  • An MVC (Model, View, & Control) approach

The MVC approach refers to how you structure your code: the model represents your data logic, the view controls what the user sees (HTML), and the controller connects both.

Installation

Laravel needs a proper development environment. Start by installing XAMPP which includes MySQL and PHP.

Step 1:

Install Composer from getcomposer.org.

Step 2:

After installing, run:

composer -V

This should return the Composer version. If not, reinstall.

Step 3:

Run:

composer global require laravel/installer

Step 4:

Create your Laravel project in your documents directory:

cd documents
cd laravel
laravel new pizzahouse

Step 5:

Navigate into the project and open with VS Code or another editor:

cd pizzahouse

In VS Code, go to Terminal → New Terminal to access the project terminal.

Start Laravel's development server:

php artisan serve

Open 127.0.0.1:8000 in your browser to see the default Laravel homepage.
