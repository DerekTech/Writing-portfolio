Creating a Database

We've discussed views, routes, and controllers. It's time to make a shift to the database side of things, so we can start incorporating some real data for our Pizzas...

Using PHPMyAdmin

Step 1: Run Xampp, start Apache and MySql servers,

Step 2: Click on the MySql admin. This will open up the PHPMyAdmin page.

Step 3: Click on New and enter pizzahouse as the database name.

Using Command Line

Step 1: Add MySQL to your PATH variable. Go to XAMPP > MySQL > bin, copy the path...

Search "env" in Start Menu:

Step 2: Login to MySQL from terminal:

mysql -u root

You should see a successful login message like:

Step 3: Create the database:

CREATE DATABASE pizzahouse;

Step 4: Configure Laravel's database connection:

DB_DATABASE=pizzahouse
DB_USERNAME=root
DB_PASSWORD=

Done! You've connected Laravel to a new database. In the next module, we’ll break down migrations so they’re easy to understand and use confidently.
